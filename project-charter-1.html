Goals

  • Goal 01

    Thiết kế Frontend để tạo ra một trang web CV digital đẹp mắt và chuyên nghiệp.

  • Goal 02

    Thiết kế web responsive, đảm bảo trang web hoạt động tốt trên laptop, tablet và smartphone.

  • Goal 03

    Đưa trang web lên internet để mọi người có thễ truy cập.

Scopes

IN

Thiết kế giao diện Frontend cho trang web bằng HTML, CSS và Bootsrap 5

Áp dụng các nguyên lý thiết kế web để tạo ra trang web đúng theo thiết kế ban đầu.

OUT

Tạo hiệu ứng animation cho trang web.

Thay đổi màu sắc, và sáng tạo theo sở thích cá nhân.
